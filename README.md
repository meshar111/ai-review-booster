# AI Review Booster

MVP عربي بسيط يساعد الأنشطة المحلية على جمع تقييمات Google من العملاء الراضين، وتحويل ملاحظات العملاء غير الراضين إلى شكوى خاصة قبل أن تصبح تقييمًا سلبيًا علنيًا.

## التشغيل

```bash
node server.js
```

ثم افتح:

```text
http://localhost:4173
```

## المزايا

- لوحة تحكم لصاحب النشاط
- صفحة تقييم للعميل
- تحويل تقييم 4 أو 5 إلى رابط Google Review
- تحويل تقييم 1 إلى 3 إلى ملاحظة خاصة
- رسالة جاهزة للعميل
- جدول ردود محفوظ في المتصفح
- تصدير CSV

## الملفات

- `index.html`: واجهة التطبيق
- `styles.css`: التنسيق
- `script.js`: منطق التطبيق
- `server.js`: سيرفر محلي بسيط
